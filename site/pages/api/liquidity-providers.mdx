# Liquidity Providers API Documentation

## Overview
The Liquidity Providers endpoint returns all the available liquidity providers given a chain id.

## Swagger

https://app.swaggerhub.com/apis/sushi-labs/sushi/5.0.0

## Endpoints

### 1. **Get Liquidity Providers**

`GET https://api.sushi.com/liquidity-providers/v5/{chainId}`

**Description:**  
Get liquidity providers for a given chainId.

**Request:**

- **Parameters:**
  - `chainId` (number, required): The chain identifier.
