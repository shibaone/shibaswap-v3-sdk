# Token API Documentation

## Overview
The Token API provides functionality to retrieve token data. Below are the endpoints available in this API.

## Swagger

https://app.swaggerhub.com/apis/sushi-labs/sushi/5.0.0

## Endpoints

### 1. **Get Token**

`GET https://api.sushi.com/token/v1/{chainId}/{address}`

**Description:**  
Get token for a given chainId and address.

**Request:**

- **Parameters:**
  - `chainId` (number, required): The chain identifier.
  - `address` (string, required): The currency address.
