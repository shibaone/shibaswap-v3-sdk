# Installation

Install Sushi via your package manager, a `<script>` tag, or build from source.

## Package Manager

Install the required packages.

:::code-group
```bash [pnpm]
pnpm add sushi
```

```bash [npm]
npm install sushi
```

```bash [yarn]
yarn add sushi
```

```bash [bun]
bun add sushi viem
```
:::

## CDN

If you're not using a package manager, you can also use Sushi via an ESM-compatible CDN such as [esm.sh](https://esm.sh). Simply add a `<script type="module">` tag to the bottom of your HTML file with the following content.

```html
<script type="module">
  import { simulateProcessRoute } from 'https://esm.sh/sushi'
</script>
```

## Usage with Viem

```ts
import { createWalletClient, http } from 'viem'
import { mainnet } from 'viem/chains'
import { publicActions, walletActions } from 'sushi'
 
const publicClient = createPublicClient({
  chain: mainnet,
  transport: http()
}).extends(publicActions)

const walletClient = createWalletClient({
  chain: mainnet,
  transport: http()
}).extends(walletActions)

const { request } = await client.simulateProcessRoute()

  .catch(console.error)
```
